## 3. 선정 주제

- 미정 (6월 4째주에 팀 회의 이후 주제 및 데이터 가공 방향 결정)



## 4. 결과물 도출 일정

- 6월 4째주 : 팀 주제 선정 및 금융 데이터 대회 접수
- 7월 1째주 : 환경 설정 및 가공 방향 MD 작성, 사용할 ML 알고리즘 지정 (문서 제작 등)
- 7월 2째주 ~ 8월 중순 : 결과물 제작
- 8월 4째주 ~ 9월 5일 : 결과물 리뷰
- 9월 6일 : 제출
